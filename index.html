Responsive Columns

Set explicit widths for more control over column width at various breakpoints.

Name Width in pixels
xs 0 - 36rem 0 - 576px
sm > 36rem > 576px
md > 48rem > 768px
lg > 62rem > 992px
xl > 75rem > 1200px

Available Widths: 20, 25, 30, 33, 40, 50, 60, 67, 70, 75, 80

Class names are structured like this:

<div class="column {breakpoint}-{width}"> or <div class="column md-33">

                    
<div class="container">
    <div class="row"></div>
        <div class="column sm-50 lg-25"></div>
            <div class="example-content"></div>
        </div>
        <div class="column sm-50 lg-25"></div>
            <div class="example-content"></div>
        </div>
        <div class="column sm-50 lg-25"></div>
            <div class="example-content"></div>
        </div>
        <div class="column sm-50 lg-25"></div>
            <div class="example-content"></div>
        </div>
    </div>
</div>

Example: Grid with 4 columns on large screens, 2 columns on middle screens, and 1 on smallest

Row Reverse

Reverse the order of columns in a row. Supports breakpoints.

                    
<div class="container">
    <div class="row"></div>
        <div class="column"></div>
            <div class="example-content"></div>
        </div>
        <div class="column"></div>
            <div class="example-content red"></div>
        </div>
    </div>
    <div class="row sm-reverse"></div>
        <div class="column"></div>
            <div class="example-content"></div>
        </div>
        <div class="column"></div>
            <div class="example-content red"></div>
        </div>
    </div>
</div>

Example: On smallest screens red is second, but the order is reversed on the second row at the first breakpoint.
